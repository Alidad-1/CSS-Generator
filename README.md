# 🎨 MagicCSS - Visual CSS Generator

MagicCSS ek interactive web tool hai jo developers ko complex CSS properties generate karne mein madad karta hai. Ismein real-time preview ka feature hai, taake aap jo tabdeeli karein woh foran dekh sakein.

## 🚀 Features

Is generator mein aap ye sab customize kar sakte hain:
- **Box & Text Shadow:** Deep shadows aur glowing effects banayein.
- **Transforms:** Rotate, Scale, aur Skew ke sath khelein.
- **Gradients:** Linear gradients design karein.
- **Flexbox & Grid:** Layouts ko visually align karein.
- **Filters:** Blur, brightness, aur contrast adjust karein.
- **Glassmorphism:** Backdrop filter ka istemal karke modern UI banayein.
- **Animations & Transitions:** Smooth motion effects define karein.
- **One-Click Copy:** Direct CSS code copy karke apne project mein lagayein.

## 🛠️ Tech Stack
- **HTML5** (Structure)
- **CSS3** (Styling & Layout)
- **JavaScript** (Logic & Real-time Updates)
